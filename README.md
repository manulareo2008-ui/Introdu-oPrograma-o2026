# Exercício da Aula de Introdução à programação 

Este repositório foi criado como parte de uma atividade para a disciplina de introdução à programação, ministrada pela professora Luciana.

#  Descrição do Projeto
O objetivo deste repositório é guardar um código simples desenvolvido durante a aula. O código resolve o problema de calcular a área de uma determinada supericie, dependendo da largura e do comprimento informados.

# Tecnologias Utilizadas
* Java
